# Campus Services Request & Tracking System

Senior Capstone Project for CIS 4891  
Team Name: Capstone Group 2

## Project Overview
This web-based system allows campus users to submit and track service requests for IT support, facilities maintenance, and general operations. The system supports role-based access for Requesters, Technicians, Managers, and Administrators.

## Key Features
- Service request submission with tracking numbers
- Category-based routing to service teams
- Technician assignment management
- Status history tracking
- Request notes and work logs
- Role-based access control
- Reporting dashboard with filtered CSV export

## Technology Stack
- ASP.NET Core MVC
- C#
- SQL Server / Azure SQL
- Bootstrap
- Entity Framework Core
